# 🌐 Plump.com Landing Page

A modern and responsive landing page built for **Plump.com**, designed to provide an engaging user experience with clean UI, smooth layout, and visually appealing sections.

---

## Project Overview

This project is a front-end landing page that showcases a product/service in a professional way. It focuses on:

- Clean and minimal design
- User-friendly interface
- Responsive layout for all devices

---

## Features

- Attractive UI/UX
- Smooth layout and section transitions
- Responsive design
- Navigation bar with proper alignment
- Hero section with call-to-action
- Footer with essential links

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Swipper.js

---

## How to Run the Project

1. Download or clone the repository
2. Open the project folder in Visual Studio Code
3. Open `index.html` in your browser
